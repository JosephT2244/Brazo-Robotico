Configuración operativa

Ajusta rangos de trabajo y una posición base para mantener una operación consistente y segura.

Límites y cero por servo — MG995 Hi-SPEED 180°
Para cada servo puedes restringir su rango operativo (mín/máx) dentro del rango físico permitido (Base 180°, Hombro 90°, Codo 45°, Muñeca 180°, Pinza 60°) y calibrar el PWM del 0° lógico para alinearlo con el cero mecánico real del servo. Los cambios se aplican en caliente.
Posición base
Coloca el brazo en la posición que deseas usar como referencia y guárdala como posición base. Esta referencia facilita volver a un punto seguro de trabajo siempre que el equipo esté alineado antes de guardarla.
